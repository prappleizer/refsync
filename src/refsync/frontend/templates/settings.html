About Citations

When you add papers, arXiv Library automatically generates BibTeX citations using arXiv metadata. These citations use the format LastName:Year (e.g., McCallum:2025).

With an ADS API key configured, you can sync citations to:

  • Detect when papers have been published in journals
  • Get complete publication metadata (journal, volume, pages, DOI)
  • Keep your BibTeX entries up to date
